¿Qué es HTML5?

Esto fue posteado el 22/07/2025 por Odalis Arana

Durante los últimos años. el estándar HTML5 se ha estado afianzando en muchas paginas web. Sin embargo. ¿qué queremos decir con esto? ¿Cuál es la diferencia crucial entre el "común", por así decirlo y este nuevo estándar? Hoy Hablaremos sobre esto y responderemos a la pregunta de qué es HTML5.

HTML5 Es un lenguaje más semántico. En otras palabras, se dota, de un significado a las diferentes partes de la página web, de forma que el navegador puede distinguir la parte que forma parte de la cabecera gracias a que hay una etiqueta que dice "cabecera", bueno lo dice en ingles: "header". ¿Para qué sirve? Ayuda a los buscadores como Google a entender tu página web, algo que se espera sea positivo de cara al SEO, y también sirve para que los navegadores muestren de forma inteligente diferentes versiones de la página Web sin tener que usar plug.ins para dispositivos específicos.

Introducción a CSS3

Esto fue posteado el 22/07/2025 por Odalis Arana

El nombre hojas de estilo en cascada viene del ingles Cascadeling Style Sheets, del que toma sus siglas. CSS es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML o XML (y por existensión en XHTML). El W3C(World Wide Web Consortium) es el encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.

¿Para que sirve el CSS3?

El CSS sirve para definir la estética de un sitio web en un documento externo y eso mismo permite que modificando ese documetno (la hoja CSS) podamos cambiar la estetica entera de un sitio web, el mismo sitio web puede variar totalmente de estética cambiando solo la CSS, sin tocar para nada los documentos HTML o jsp o asp que lo componen.
